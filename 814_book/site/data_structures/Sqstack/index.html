    顺序栈

    顺序栈是指利用顺序存储结构实现的栈，即利用一组地址连续的存储单元依次存放自栈底到栈顶的数据元素，同时附设指针 top 指示栈顶元素在顺序栈中的位置。通常习惯的做法是：以 top = 0 表示空栈，鉴于 C 语言中数组的下标约定从 \(0\) 开始，则当以 C 语言作描述语言时，如此设定会带来很大不便，因此另设指针 base 指示栈底元素在顺序栈中的位置1。当 top 和 base 的值相等时，表示空栈。

    算法步骤

    1. 为顺序栈动态分配一个最大容量为 \(MAXSIZE\) 的数组空间，使 base 指向这段空间的基地址，即栈底；
    2. 栈顶指针 top 初始为 base，表示栈为空；
    3. stacksize 置为栈的最大容量 \(MAXSIZE\)。

int top[2] = {0, MAX - 1};
int S[MAX];
void push(int I, int X)
{
    if (top[0] + 1 == top[1]) return;
    S[top[I]] = X;
    top[I] += I == 0? 1 : -1;
}

    算法设计题

    假设表达式中允许包含三种括号：圆括号()、方括号[] 和大括号{}。请简述实现判断表达式中的括号是否正确配对的算法思想。

    算法思想：

    若匹配到左括号，直接入栈；若匹配到右括号，则从栈顶弹出一个左括号，若弹出的左括号与当前右括号匹配，则匹配成功，否则匹配失败。

    bool isMatch(char *exp)
{
    char *Stack = new char[strlen(exp) + 1];
    int top = 0;
    for (int i = 0; exp[i]; i++)
    {
        if (exp[i] == '(' || exp[i] == '[' || exp[i] == '{')
        {
            Stack[top++] = exp[i];
        }
        else if (exp[i] == ')' || exp[i] == ']' || exp[i] == '}')
        {
            if (top == 0) return false;
            if (exp[i] == ')' && Stack[top - 1] == '(') top--;
            else if (exp[i] == ']' && Stack[top - 1] == '[') top--;
            else if (exp[i] == '}' && Stack[top - 1] == '{') top--;
            else return false;
        }
    }
    return top == 0;
